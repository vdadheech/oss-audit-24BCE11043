## 🛠️ Scripts Overview

| File                               | Purpose                                                                                       |
| ---------------------------------- | --------------------------------------------------------------------------------------------- |
| script1_system_identity.sh         | Displays kernel version, current user, system uptime, distribution, and OS license            |
| script2_package_inspector.sh       | Checks if the Linux kernel is installed, shows version details, and includes philosophy notes |
| script3_disk_permission_auditor.sh | Audits important directories for permissions and disk usage                                   |
| script4_log_analyzer.sh            | Analyzes a log file, counts keyword matches, and shows last 5 matches                         |
| script5_manifesto_generator.sh     | Generates a personalized open-source philosophy statement                                     |

## ⚙️ How to Run

### Prerequisites

* Linux system (Ubuntu 22.04 / Debian / Red Hat based)
* Bash shell (pre-installed)
* No additional dependencies required

### Steps

1. Clone the repository:
   git clone [https://github.com/vdadheech/oss-audit-24BCE11043](https://github.com/vdadheech/oss-audit-24BCE11043)

2. Navigate into the folder:
   cd oss-audit-24BCE11043

3. Make scripts executable:
   chmod +x *.sh

4. Run the scripts:
   ./script1_system_identity.sh
   ./script2_package_inspector.sh
   ./script3_disk_permission_auditor.sh
   ./script4_log_analyzer.sh /var/log/kern.log error
   ./script5_manifesto_generator.sh

## 📊 Project Description

This project performs a basic audit of the Linux Kernel using Bash scripting. Each script focuses on a specific system component including system identity, package inspection, disk permissions, log analysis, and open-source philosophy.

It demonstrates practical usage of shell scripting for system monitoring and introduces key open-source concepts.
